Votre messagerie Jagex

Vous pouvez consulter le statut de vos questions en cliquant sur Lire les messages dans la section Gestion du compte. Vous pouvez également vous connecter pour consulter vos rappels de paiement et autres avis que Jagex vous envoie régulièrement. L'écran qui s'affiche est le suivant :



La boîte de réception effectue le suivi de vos communications avec le personnel de Jagex en les répartissant dans trois dossiers :

  • Reçus
  • Envoyés
  • Lus


Tout message figurant dans le dossier Reçus est un nouveau message que Jagex vous adresse. Il est préférable de l'ouvrir immédiatement.

Les messages figurant dans Envoyés sont les messages que vous avez envoyés en réponse à Jagex ou vos messages spontanés.

Vous pouvez remarquer la présence de deux boutons à la fin de chaque conversation :
Ce bouton ouvre le message pour que vous puissiez le lire.
Ce bouton supprime l'intégralité de la conversation. Vous pouvez toujours supprimer des messages du dossier Lus et du dossier Envoyés tant qu'il n'ont pas été assignés à un membre du personnel. Cela peut être utile si vous avez trouvé la réponse à votre question avant que Jagex vous envoie la réponse.


Envoi d'une réponse


Lorsque vous ouvrez un message (en cliquant sur l'objet ou sur l'icône de lecture située à droite du message), vous êtes directement redirigé sur une page de ce type :



Comme vous pouvez le constater, cette page contient tous les messages provenant de Jagex et de vous-même sur ce sujet. Si vous souhaitez envoyer une réponse, vous pouvez cliquer sur Répondre (bouton jaune situé au-dessous du message le plus récent). Une boîte de message dans laquelle vous pouvez saisir votre message s'affiche. Cliquez ensuite sur Envoyer. Notez que votre message ne doit pas dépasser 400 caractères.

Si le lien Répondre n'apparaît pas, cela signifie que Jagex n'a pas jugé utile que vous répondiez. Si vous souhaitez poser des questions, vous devez débuter une nouvelle conversation en envoyant une nouvelle question.

Jagex ne répondra pas forcément à tous vos messages car il se peut que nous résolvions votre problème sans juger utile de répondre. Par conséquent, ces messages peuvent être déplacés du dossier Envoyés au dossier Lus.

Vous pouvez vous connecter à votre boîte de réception à partir de la page Gestion du compte.
