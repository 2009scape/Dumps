1. Comment signaler un bug ?

Vous pouvez signaler un bug en cliquant ici.

Essayez de fournir le maximum d'informations par l'intermédiaire du formulaire afin de nous aider à repérer et corriger le bug le plus rapidement possible. Dès réception de votre rapport, un membre du personnel de Jagex effectuera une recherche. Sachez que Jagex ne récompense pas les joueurs qui signalent des bugs.

Signalez uniquement des bugs que vous avez vous-même rencontrés et non des bugs décrits par un ami ou dans les forums.

Un rapport de bug doit comporter le maximum d'informations détaillées pour nous aider à reproduire le bug. Le simple fait de préciser le jeu et le niveau auxquels vous jouiez ou les touches sur lesquelles vous avez appuyé peut être déterminant.

Voici un exemple de rapport peu utile : « Je n'arrive pas à terminer le niveau 2. »

Un rapport plus utile serait du type suivant : « Je n'arrive pas à terminer le niveau 2 de Kristal Folie car les quatre rochers empilés ne bougent pas, même si j'essaie de les pousser ; je ne peux donc pas atteindre les tout derniers cristaux. »

Le formulaire Rapport de bug ne doit servir que pour les bugs des jeux ou du site Web. Les problèmes de facturation, techniques ou les commentaires sur les publicités doivent être soumis par le biais des liens suivants : Facturation, Assistance technique, Publicité.

Si vous n'arrivez pas à trouver de solution à votre problème à ce stade, contactez le service correspondant pour obtenir une aide supplémentaire. Nous ignorerons tout rapport de bug qui n'est pas strictement lié à un bug. Pour en savoir plus sur la définition d'un bug, lisez la question 3 ci-dessous.

2. Que faire si je vois quelqu'un profiter d'un bug pour tricher ?

Si vous repérez quelqu'un qui se sert d'un bug pour tricher, veuillez nous le signaler en cliquant ici afin que nous puissions examiner la situation et agir de manière appropriée quant au bug et aux joueurs qui l'exploitent. Notez que Jagex ne peut garantir une réponse à chaque rapport de bug.

3. Qu'est-ce qu'un bug ?

Un bug est une erreur située dans le code d'un jeu et qui nuit à son exécution. En d'autres termes, un bug est un morceau de code erroné qui empêche un jeu de fonctionner correctement (par exemple, votre personnage ne peut plus sauter dans un jeu de plate-forme).

Si vous pensez avoir détecté un bug, vous pouvez le signaler en cliquant ici. Si vous trouvez un bug qui vous permet de tricher en le renouvelant, ne l'exploitez pas. L'exploitation de bug est interdite et les joueurs qui la pratiquent seront exclus.

4. Qu'est-ce que l'exploitation de bug ?

Une exploitation de bug a lieu lorsque un joueur a trouvé un bug dans l'un de nos jeux et l'exploite à des fins personnelles. Lorsqu'un bug a été trouvé et peut être reproduit pour tricher, veuillez le signaler en cliquant ici.

Notez que l'exploitation de bug est contraire au règlement de FunOrb. Consultez les règles de conduite pour en savoir plus. Tout joueur pris en train d'exploiter un bug pour tricher sera sanctionné.

5. Dans quelles conditions dois-je signaler un bug ?

Ne soumettez pas de rapport de bug si vous souhaitez simplement contacter Jagex.

N'envoyez pas non plus de rapport si vous êtes simplement bloqué dans un jeu FunOrb. Faites une pause et rejouez plus tard. Vous pouvez également demander des conseils aux autres joueurs dans les forums.

Ne soumettez un rapport de bug que si vous êtes certain d'avoir essayé toutes les autres possibilités.

Jagex sanctionnera tout joueur qui utilise le système de signalement de bug de manière abusive.
