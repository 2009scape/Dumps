Le jeu ne se charge pas

  1. Je peux accéder à l'écran de jeu principal mais celui-ci affiche uniquement "Chargement en cours... Veuillez patienter". Que dois-je faire ?
  2. Je n'arrive pas à charger un jeu FunOrb. Même l'écran-titre ne se charge pas ! Que dois-je faire ?
  3. Les jeux se bloquent avant même de se charger. Que dois-je faire ?
  4. Le site Web se charge très lentement. Que dois-je faire ?

1. Je peux accéder à l'écran de jeu principal mais celui-ci affiche uniquement "Chargement en cours... Veuillez patienter". Que dois-je faire ?

Si votre ordinateur dispose d'1 Go minimum de RAM, il est possible qu'un conflit de mémoire existe. En d'autres termes, Java ne fournit pas assez de mémoire au jeu FunOrb. Pour résoudre ce problème, procédez comme suit.

Remarque : ne réalisez cette procédure que si vous possédez les compétences techniques adéquates ou êtes un adulte responsable. Jagex décline toute responsabilité en cas de perte de données à la suite de cette procédure de dépannage.

- Ouvrez le Panneau de configuration Windows.
- Ouvrez le panneau de configuration Java.
- Sélectionnez l'onglet Java.
- Dans la section Paramètres de l'applet Java Runtime, sélectionnez Afficher... (veillez à bien afficher les paramètres de l'applet et non de l'application).

Votre version de JRE doit apparaître dans la liste. Dans la zone des paramètres d'exécution, vous pouvez saisir une ligne du type :

-Xmx256M
Saisissez bien le signe moins en début de ligne. Cela devrait accroître la mémoire maximum disponible pour Java et la définir à 256 Mo.
Ne définissez pas un chiffre supérieur à la moitié de votre RAM maximum sous peine de créer des problèmes d'exécution de Windows.

Les jeux FunOrb étant conçus pour être exécutés sous Windows à l'aide du navigateur Internet Explorer, nous pouvons uniquement offrir de l'assistance sur ces produits. Si vous utilisez un système d'exploitation ou un navigateur différent, reportez-vous au manuel de votre produit pour connaître les procédures équivalentes.

Si les problèmes persistent, contactez l'équipe du service clientèle pour obtenir une assistance supplémentaire. Pensez à nous communiquer tous les changements qui se sont produits sur les performances du jeu suite aux actions ci-dessus, ainsi que les spécifications de votre ordinateur (puissance du processeur, connexion Internet, RAM, espace libre sur le disque dur).

2. Je n'arrive pas à charger un jeu FunOrb. Même l'écran-titre ne se charge pas ! Que dois-je faire ?

Veuillez effectuer les différentes actions ci-dessous dans l'ordre pour résoudre votre problème technique. Si la première action ne résout pas le problème, passez à la suivante.

  • Essayez de vous connecter à un autre serveur. La liste des serveurs du jeu se trouve à la page d'informations, juste au-dessus du bouton Description. Si l'un de nos serveurs connaît des problèmes techniques, cela résoudra le problème jusqu'à ce que nous le réparions.

  • Il se peut que votre navigateur mette en cache une version antérieure du jeu. Essayez de vider votre cache Web. Dans Internet Explorer, dans la barre des options, sélectionnez Outils > Options Internet > Fichiers Internet temporaires > Supprimer les fichiers...

  • Dans le menu déroulant Jeux du site Web FunOrb, sélectionnez Options Java. Cliquez sur la flèche déroulante du menu affiché et sélectionnez Applet signé par Sun Java puis réessayez de charger le jeu normalement. Cela résout généralement le problème.

  • Assurez-vous d'avoir accepté l'avertissement de sécurité figurant dans la fenêtre contextuelle qui s'affiche quelques secondes après la sélection de votre serveur de jeu. Si la fenêtre contextuelle ne s'affiche pas, essayez de redémarrer votre ordinateur et cliquez bien sur Oui lorsque vous êtes invité à autoriser l'applet FunOrb.

  • Dans le menu déroulant Jeux du site Web FunOrb, sélectionnez Options Java. Cliquez sur la flèche déroulante du menu affiché et sélectionnez Applet Java non signé puis réessayez de charger le jeu normalement. Cela résout généralement le problème si vous jouez sur un ordinateur public sur lequel vous disposez d'un accès limité.

  • Téléchargez Mozilla FireFox sur www.mozilla.org et essayez de l'utiliser à la place d'Internet Explorer pour exécuter les jeux FunOrb.

  • Il est possible que nos serveurs soient temporairement hors ligne à des fins de maintenance ou de mise à jour. Patientez une heure et réessayez.

  • Assurez-vous que votre ordinateur est assez puissant pour exécuter le jeu en question. Chaque jeu FunOrb possède sa propre configuration minimale requise, que vous pouvez consulter sur la page d'informations du jeu. Si votre ordinateur correspond plus ou moins à la configuration requise spécifiée, veillez à fermer tous les programmes inutiles avant d'exécuter le jeu.


Si les problèmes de chargement persistent, contactez l'Assistance technique.

3. Les jeux se bloquent avant même de se charger. Que dois-je faire ?

Si vous essayez d'exécuter un jeu FunOrb sur un ordinateur équipé de Windows Vista et d'Internet Explorer, il se peut que Windows soit configuré en mode protégé. Ce mode n'autorise pas la mise en cache locale des jeux FunOrb et empêche donc leur chargement. Pour résoudre ce problème, procédez comme suit :

- Dans Internet Explorer, sélectionnez Outils > Options Internet > Sécurité > Sites de confiance > Sites... - Désactivez la case à cocher Nécessite un serveur sécurisé (https:) pour tous les sites dans cette zone. - Saisissez *.funorb.fr dans le volet de saisie Ajouter ce site Web à la zone et cliquez sur Ajouter.

Sinon, vous pouvez télécharger le navigateur Web Mozilla sur www.mozilla.org et essayer de l'utiliser à la place d'Internet Explorer pour exécuter les jeux FunOrb.



Si les problèmes de chargement persistent, contactez l'Assistance technique.

4. Le site Web se charge très lentement. Que dois-je faire ?

Notre site Web met parfois du temps à se charger après une mise à jour car beaucoup de personnes essaient de télécharger les nouveaux jeux simultanément. Patientez quelques instants et réessayez de charger le site Web.
