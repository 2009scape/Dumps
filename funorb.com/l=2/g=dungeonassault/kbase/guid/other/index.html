1. Comment soumettre une question technique ?

Avant d'envoyer un formulaire d'assistance technique, veuillez vérifier que votre ordinateur est mis à jour en suivant les instructions suivantes :

Mise à jour de Windows


Pour télécharger les correctifs de mise à niveau pour le système d'exploitation Windows, rendez-vous sur le Centre de téléchargement de Windows sur http://windowsupdate.microsoft.com/


Mise à jour de Java


La machine virtuelle utilisée pour exécuter les jeux FunOrb devra être mise à niveau vers toutes les nouvelles versions qui sont développées et mises à la disposition des utilisateurs. Ces mises à niveau se trouvent sur la page d'accueil de Sun Java à l'adresse suivante : java.com/fr/download/index.jsp


Mise à jour de votre navigateur Web (ex. : Internet Explorer, Firefox)


Quel que soit le navigateur Web que vous utilisez, nous vous recommandons d'utiliser la dernière version afin de garantir une exécution et une sécurité Internet optimales.

Pour mettre à jour votre navigateur Web, rendez-vous sur le site Web officiel du développeur ou utilisez le lien adéquat fourni par le navigateur.


Une fois votre ordinateur mis à jour, cliquez ICI pour continuer.


2. Comment accéder à mes informations système ?

Localisation des informations système de votre système d'exploitation Windows


Pour localiser les informations logicielles de votre système ainsi que les informations relatives à votre matériel, procédez comme suit :
  1. Menu Démarrer > Paramètres > Panneau de configuration
  2. Sélectionnez l'icône Système.
  3. L'onglet Général contient des informations indiquant la version de Windows, la quantité de RAM dont vous disposez et la puissance de votre processeur.

Localisation de votre version Java


Pour localiser les informations logicielles de Java (uniquement si Sun Java est installé) :
  1. Menu Démarrer > Paramètres > Panneau de configuration
  2. Sélectionnez l'icône Java.
  3. Cliquez sur le bouton À propos de... pour afficher la version Java que vous utilisez.

Localisation des informations système de votre système d'exploitation Mac
  1. Cliquez sur l'icône Apple bleue située dans l'angle supérieur gauche de votre écran et sélectionnez À propos de ce Mac.
  2. La fenêtre qui s'affiche contient la version du système d'exploitation ainsi que les informations liées au processeur et à la mémoire de votre Mac.

Localisation des informations système de votre navigateur
  1. Ouvrez votre navigateur (Internet Explorer, Firefox, etc.).
  2. Sélectionnez le menu d'aide et sélectionnez À propos de. Les informations liées à la version de votre navigateur s'affichent.

3. La fenêtre de jeu FunOrb peut-elle être redimensionnée ?

Si vous êtes un abonné FunOrb, vous pouvez effectivement bénéficier du mode plein écran. Pour les non-abonnés, ce mode n'est pas disponible.

Le mode plein écran nécessite l'utilisation de l'applet signé (il s'agit de l'option par défaut ; si vous souhaitez vérifier ou modifier cela, sélectionnez Options Java dans le menu Jeux en haut de la page). Nous nous efforçons de prendre en charge ce mode sur le plus d'ordinateurs possibles ; cependant, il existera toujours des ordinateurs sur lesquels il ne fonctionnera pas. Si vous rencontrez des difficultés, mettez à niveau votre version Java ou vos pilotes graphiques.

4. Comment désactiver le plug-in SUN Java ?

Dans une fenêtre Internet Explorer, sélectionnez Outils > Options Internet dans la barre d'options. Dans la boîte de dialogue qui s'affiche, sélectionnez l'onglet Avancé et accédez à la section Java (Sun). Désactivez la case à cocher pour désactiver le plug-in Sun Java.

5. Quels sont les navigateurs pris en charge ?

FunOrb prend en charge la plupart des navigateurs. Pour optimiser votre jeu, nous vous recommandons l'un des navigateurs suivants :

Microsoft Internet Explorer 6 ou version ultérieure.

Mozilla Firefox 1.5 ou version ultérieure

Apple Safari.

6. Le jeu s'exécute lentement. D'où cela provient-il ?

S'il s'agit de votre première utilisation du jeu FunOrb en question ou si nous avons mis à jour le jeu depuis votre dernière utilisation, le chargement peut être plus long que d'habitude car aucune donnée de jeu n'est mise en cache sur votre ordinateur. Si le problème n'est pas dû à cela, notez les points suivants :

Une exécution de jeu lente est généralement causée par un ordinateur lent ou une connexion Internet faible entre votre ordinateur et notre serveur. Même si vous utilisez un ordinateur super puissant connecté à l'aide d'un câble ultra rapide, il est possible que l'un des ordinateurs du réseau qui nous relie ne soit pas suffisamment performant et ralentisse parfois votre exécution.

Pour réduire le problème, vous pouvez effectuer la procédure suivante :

  • 1. Essayez de jouer sur un autre serveur. La liste des serveurs est accessible par le biais des pages d'informations, juste au-dessus du bouton Description.

  • 2. Assurez-vous qu'aucun autre programme n'utilise votre connexion Internet. Fermez tous les logiciels inutiles en cours d'exécution sur votre ordinateur. De même, évitez de jouer aux jeux FunOrb en même temps que vous téléchargez des fichiers ou regardez la télé sur Internet.

  • 3. Essayez de jouer à un autre moment de la journée, lorsque le Web est plus calme et que les divers routeurs Internet ne sont pas surchargés.

  • 4. Dans le menu déroulant Jeux du site Web FunOrb, sélectionnez Options Java. Cliquez sur la flèche déroulante du menu affiché et sélectionnez Applet signé à l'aide de Sun Java puis réessayez de charger le jeu normalement.

  • 5. Si vous utilisez une version antérieure de Windows, assurez-vous que votre compilateur Java Just-In-Time est activé pour la machine virtuelle. Dans Internet Explorer, sous Outils > Options Internet > Avancé > Microsoft VM.

  • 6. Essayez d'améliorer les performances de votre ordinateur.

  • 7. Essayez avec un autre fournisseur d'accès à Internet.


Nos serveurs disposent d'une très bonne liaison Internet ; il est donc peu probable que le problème vienne de notre côté.
