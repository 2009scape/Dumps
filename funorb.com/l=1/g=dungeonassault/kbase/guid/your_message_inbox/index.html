Ihr Mitteilungsfach

Den Status Ihrer Anfragen können Sie überprüfen, indem Sie sich vom Hauptmenü Kontoverwaltung in Ihr Mitteilungsfach einloggen und zwar über den Link 'Nachrichten lesen'. Sie können sich dort auch einloggen, um eventuelle Zahlungserinnerungen und andere Mitteilungen einzusehen, die Jagex in gewissen Abständen verschickt. Die Seite sieht dann so aus:



Ihr Mitteilungsfach gibt Ihnen einen Überblick über die Nachrichten, die Sie mit Jagex ausgetauscht haben. Sie werden in drei Gruppen unterteilt:

  • Erhalten
  • Abgeschickt
  • Gelesen


Wenn sich im Ordner 'Erhalten' eine Nachricht befindet, heißt das, dass Jagex Ihnen eine neue Mitteilung geschickt hat. Sie sollten sie umgehend lesen.

Im Ordner 'Abgeschickt' befinden sich die Nachrichten, die Sie beantwortet haben oder Anfragen, die Sie abgeschickt, zu denen Sie aber noch keine Antwort erhalten haben.

Am Ende jedes Dialogs gibt es zwei weitere Schaltflächen:
Mit dieser Schaltfläche wird die Mitteilung geöffnet, sodass Sie sie lesen können.
Mit dieser Schaltfläche wird der ganze Dialog gelöscht. Im Ordner 'Gelesen' können Sie immer Mitteilungen löschen und Sie dürfen auch Mitteilungen im Ordner 'Abgeschickt' löschen, wenn sie noch keinem Mitarbeiter zugewiesen wurden. Das sollten Sie beispielsweise tun, wenn Ihnen die Antwort zu Ihrer Anfrage schon eingefallen ist. So können Sie Jagex eine Antwort ersparen.


Eine Antwort abschicken


Wenn Sie eine Mitteilung öffnen (indem Sie auf das Symbol 'Lesen' rechts von der Mitteilung klicken), werden Sie zu einer Seite wie dieser weitergeleitet:



Hier werden Ihnen alle Mitteilungen in diesem Dialog angezeigt, die von Ihnen und von Jagex verfasst wurden. Wenn Sie es für nötig halten, eine Antwort zu verfassen, können Sie auf 'Antwort schicken' klicken (dieser Link ist grün gekennzeichnet und befindet sich unter der neuesten Mitteilung). Dann wird ein Textfeld für Sie geöffnet, in das Sie Ihre Mitteilung eingeben können. Dann müssen Sie nur noch auf 'Abschicken' klicken. Bitte beachten Sie, dass Ihre Mitteilungen nicht mehr als 400 Zeichen umfassen dürfen.

Falls der Link für 'Antwort abschicken' nicht vorhanden ist, hält Jagex es nicht für nötig, dass Sie auf diese Mitteilung antworten. Wenn Sie weitere Fragen an Jagex haben, müssen Sie einen neuen Dialog starten, indem Sie eine neue Anfrage abschicken.

Sie werden nicht auf alle Nachrichten eine persönliche Antwort von Jagex erhalten, da wir manchmal Ihr Problem beheben oder Ihre Anfrage lösen können, ohne dass eine Antwort wirklich nötig ist. In solchen Fällen werden Ihre Mitteilungen aus Ihrem Ordner 'Abgeschickt' in den Ordner 'Gelesen' verschoben.

Über Kontoverwaltung können Sie sich in Ihr Mitteilungsfach einloggen.
